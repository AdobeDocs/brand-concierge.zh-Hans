# 了解数据流ID

了解数据流ID如何将您网站上的Web SDK事件连接到Adobe Experience Platform中的Adobe Brand Concierge。 您可以了解为什么需要ID，它如何将数据路由到正确的沙盒和数据集，以及团队如何使用单独的数据流进行开发、暂存和生产。

* 数据流如何将事件路由到沙盒、数据集、架构和标识
* 为什么团队在开发、暂存和生产中使用不同的数据流
